Page Cypher

An Ottendorf Cypher tool using web pages as source material

The Ottendorf Cypher is a simple substitution cypher that uses a book or other text as the key. Normally, each letter in the message you want to encode is encoded as a page, word and letter from the key text.

These bookmarklets let you use the text of a web page as the key text. You can then encode or decode messages.

Cypher Encode Cypher Decode

You will need to drag the bookmarklets to your bookmarks bar. Then, when you are on a page you want to use as the key text, click the "Cypher Encode" bookmarklet to encode text. The coded message will be displayed at the top of the page.

To decode a message, click the "Cypher Decode" bookmarklet. The decoded message will be displayed in a pop-up window.
